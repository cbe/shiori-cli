# shiori-cli

Manage your [shiori](https://github.com/go-shiori/shiori) bookmarks conveniently from the CLI

## Available features

| Feature          | State |
| ---------------- | :---: |
| Log in           |  ✅   |
| Log out          |  ❌   |
| Get bookmarks    |  ❌   |
| Add bookmarks    |  ❌   |
| Edit bookmarks   |  ❌   |
| Delete bookmarks |  ❌   |
| Get tags         |  ❌   |
| Rename tags      |  ❌   |
